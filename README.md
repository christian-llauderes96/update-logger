🚀 DevLog System
A streamlined internal logging platform built with Laravel 11 and Filament v3. Designed for development teams to track system updates, monitor platform status, and generate developer activity reports.

✨ Key Features
Centralized Logging: Track updates across multiple systems/platforms.

Personalized Dashboard: Developers see their own contributions; Admins see a global overview.

Smart Reporting: Filter updates by date range and developer ownership.

Security First: Role-based access control (Admin vs. Developer) and secure password hashing.

Sleek UI: Powered by Filament, featuring a responsive dark/light mode and Heroicons.

🛠️ Tech Stack
Framework: Laravel 11

Admin Panel: Filament v3

Database: MySQL / MariaDB

Icons: Heroicons
